This is a Java-based console application that allows students to submit issues and enables tracking and updating their status.

Features:
Submit a new issue,
View all submitted issues,
Update issue status,
Priorities issues,
Search issue by issue Id,
Simple menu-driven interface

Technologies Used:
Java
OOP Concepts (Classes, Objects, ArrayList)

How to Run:
Make sure all files are in the same folder:
Main.java
IssueManager.java
Issue.java
Compile all files:
javac *.java
Run the program:
java Main

Project Structure:
Main.java → Entry point of the program (contains main method)
IssueManager.java → Handles issue operations (add, view, update)
Issue.java → Represents the issue data

Sample Flow:
User selects option from menu
Enters issue details
System stores and displays issues
User can update status using Issue ID

Future Improvements:
